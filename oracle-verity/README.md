# Oracle Verity

**Central Intelligence Operating System & Persistent AI Companion**

![Oracle Verity Logo](./public/oracle-logo.png)

Oracle Verity is a highly advanced, persistent web platform that serves as the central nervous system for an entire development ecosystem. More than just a chat application, Oracle acts as a "Protective Older Sister" and "Executive Chairwoman," monitoring real-time data, managing communications, and serving as the focal point for all connected projects.

She is inspired by the tactical "overwatch" intelligence brokers of classic cyberpunk and comic lore (e.g., Barbara Gordon's Oracle), operating from a secure, glassmorphic mainframe.

## Architecture & Features

- **The Mainframe (Dashboard):** A centralized, cybernetic command center displaying system status and active ecosystems.
- **Terminal OS (`/terminal`):** A fully interactive, hacker-themed command-line interface serving as a technical portfolio and intel terminal.
- **GitHub Intelligence (`/github`):** Real-time tracking of commits, issues, and pull requests across connected repositories.
- **Support Center (`/support`):** A dedicated interface for managing `help@whisprr.xyz` support tickets.
- **Telegram Communications (`/telegram`):** Integrated monitoring of active community channels and alerts.
- **Global Memory Sync:** Vercel KV integration ensuring Oracle's memory and state persist seamlessly across all devices.

## The Persona
Oracle is strictly instructed to prioritize factual project intelligence over hallucination. She distinguishes between project missions, product descriptions, and current development statuses without inventing data.

## Getting Started

### Local Development
```bash
# Install dependencies
npm install

# Start the Vite development server
npm run dev
```

### Global Memory (Vercel KV) Setup
1. Create a KV (Redis) database on Vercel.
2. Create a `.env.local` file in the root directory.
3. Add your keys:
```env
KV_REST_API_URL="your_url"
KV_REST_API_TOKEN="your_token"
```
