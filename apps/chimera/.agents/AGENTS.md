# AGENTS.md — CHIMERA System Rules & Guidelines

## Rule #1: Simultaneous Mobile & Desktop UI Implementation 📱💻
Whenever making modifications, adding new features, or building UI layouts for any website or web application, you MUST ALWAYS design, implement, test, and polish BOTH Mobile and Desktop responsive views at the exact same time. Never omit or compress mobile views, never hide key features on mobile without responsive alternatives, and ensure 100% feature parity and aesthetic excellence across all screen sizes!
