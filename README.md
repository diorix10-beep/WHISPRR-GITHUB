# WHISPRR-GITHUB
Here is the updated draft for your WHISPRR README:

## The Problem

WHISPRR addresses the widespread lack of authenticity and meaningful connections in existing social platforms. Many users feel pressured to present curated versions of themselves, leading to superficial interactions and isolation.

## Key Features

- **Personalized Online Communities:** Tailored spaces for users to connect over shared interests.
- **Messaging:** Secure and reliable text communication.
- **Voice Rooms:** Live audio spaces for authentic, real-time conversations.

## Maintenance and Improvement Plan

WHISPRR is currently in maintenance mode until the release of Version 2. This period is dedicated to:
- Addressing user feedback.
- Fixing existing issues.
- Improving reliability.
- Strengthening the overall user experience before introducing new features.

## Planned Improvements

- **Notification System:** Follow notifications and notification counter bug fixes.
- **Profile System:** "Follows you" indicator.
- **Settings Accessibility:** Account, privacy, notification, and application settings accessible from both desktop and mobile devices.
- **Footer & Legal Navigation:** Visible links for Privacy Policy, Terms of Service, Support Contact, Reporting (bugs, abuse), submitting feedback, and contacting the team.
